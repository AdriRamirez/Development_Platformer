# Credits
Made by Miguel Rodríguez & Adrián Ramírez         
Team name: Knights of the square table


## Features

One level         
Parallax         
Basic Movement         
Double Jump         
Debugging functions         

## Controls

A - Move Left         
D - Move Right         
W - Move Up (in god mode)         
S - Move Down (in god mode)         

Debugging options:         
F1 - Show/Hide colliders         
F5 - Save         
F6 - Load         
F10 - God Mode
